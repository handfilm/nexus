================================================================================
                      TROUBLESHOOTING
================================================================================

PROBLEM:     Portal doesn't load
SYMPTOMS:    Blank page, 404 error
SOLUTIONS:   
  □ Check file paths in index.html (use ./)
  □ Verify all 5 .js files in same directory
  □ Check browser console (F12)
  □ Try hard refresh (Ctrl+Shift+R)

PROBLEM:     Scripts don't work
SYMPTOMS:    No theme toggle, estimator broken
SOLUTIONS:
  □ Check JavaScript errors in console
  □ Verify file paths are relative
  □ Ensure GitHub Pages is enabled
  □ Wait 1-2 minutes for build

PROBLEM:     Theme doesn't persist
SYMPTOMS:    Theme resets on refresh
SOLUTIONS:
  □ Enable localStorage in browser
  □ Clear browser cache
  □ Disable private/incognito mode
  □ Check localStorage quota

PROBLEM:     Telemetry not ingesting
SYMPTOMS:    "SCHEMA_VALIDATION_FAIL" error
SOLUTIONS:
  □ Verify all 6 required fields present
  □ Check JWT token validity
  □ Look at error queue in terminal
  □ Test with NEXUS.ingestTelemetry()
